# Portfolio

Personal portfolio built with **React**, **Vite**, and **Tailwind CSS** — inspired by [ramx.in](https://ramx.in/).

## Live

**[shravanramakunja.in](https://www.shravanramakunja.in/)**

## Stack

- **React 19** + **Vite 6**
- **Tailwind CSS 4**
- **GSAP** & **Framer Motion** — animations
- **OGL** & **Three.js** — 3D graphics
- **Last.fm API** — currently playing / recently played

## Development

```bash
npm install
npm run dev     # local server at localhost:5173
npm run build   # production build
npm run preview # preview production build
```
